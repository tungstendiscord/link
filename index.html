Link

The Discord Bot for Everything Outside Discord



What is Link?

Link is a Discord bot that uses many APIs. You can get a lot of information from these APIs without leaving Discord!

How do I use Link?

The main syntax is l!<api><command>. A list of all the APIs we use are found here.

How do I support Link?

Link doesn't have a Patreon or PayPal set up, but you can support by chatting in the support server and applying to become a mod.

What if I see a bug?

You can report it to the support server.

What if I have a feature or some code I want to add to Link?

The GitHub repository is here, you can open a pull request. Feature requests can be submitted in the Support Server.

How do I apply to be a Moderator?

To be a moderator, you need to apply here.

What if I want to be part of the Dev Team?

As of now, you need to contact me directly via Discord. I change my username a lot, and it would be a hassle to keep updating it here, so you can join the support server and DM Tungsten from there.
